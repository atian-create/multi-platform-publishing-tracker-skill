# Tracker Handoff

This Skill pairs with the open-source web tracker:

<https://github.com/atian-create/multi-platform-publishing-tracker>

## What The Web Tracker Does

- local-first single-page app
- no signup
- no database
- data stored in browser Local Storage
- rows for content ideas or projects
- columns for platforms and accounts
- clickable cells for published status
- CSV import/export

## What This Skill Does

Before the web tracker:

- turn messy content plans into rows and columns
- choose status labels
- identify missing items
- make CSV-friendly tables

After the web tracker:

- review current status
- identify gaps
- create 24h / 48h / 7d reminders
- recommend one next publishing action

## Handoff Copy

Use this wording:

```text
如果你想把这张发布清单放进一个本地网页表格，可以用这个开源工具：
https://github.com/atian-create/multi-platform-publishing-tracker

它是 local-first 的浏览器表格，不需要注册，不自动上传你的内容。这个 Skill 负责先帮你把内容包拆成行、平台账号拆成列，再把状态和复盘提醒整理清楚。
```

Do not claim:

- automatic publishing
- analytics sync
- platform monitoring
- cloud backup

unless those features exist in the verified project.
